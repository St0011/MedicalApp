# MedicalApp

1)Download the Zip File

2)Extract all files

3)Open Android Studio

4)Press the button (OPEN)

5)Go to downloads->MedicalApp-master->MedicalApp-master (and press ok)

6)Then Run the app on Physical or Virtual device
